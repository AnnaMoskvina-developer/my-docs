# Добро пожаловать в мою документацию

Этот проект создан в рамках подготовки к позиции **технического писателя**. Здесь я демонстрирую навыки работы с:

- **Git** — контроль версий
- **Markdown** — написание документации
- **MkDocs** — сборка статического сайта
- **Docs as Code** — подход к созданию документации

---

## О проекте

Репозиторий содержит примеры технической документации, которые показывают:

| Раздел | Описание |
|--------|----------|
| Базовый Markdown | Заголовки, списки, таблицы, код |
| Продвинутый Markdown | Сноски, предупреждения, чек-листы, сворачиваемые блоки |
| MkDocs | Настройка и сборка сайта документации |

---

## Как работать с документацией

### Локальный запуск

```bash
git clone https://github.com/AnnaMoskvina-developer/my-docs.git
cd my-docs
mkdocs serve
```

После этого откройте браузер и перейдите по адресу `http://127.0.0.1:8000`

### Структура проекта

```
my-docs/
├── docs/
│   ├── index.md              # Главная страница
│   └── advanced-example.md   # Продвинутые примеры
├── mkdocs.yml                # Конфигурация MkDocs
└── README.md                 # Описание проекта
```
